WHAT IT DOES
================================================================================
✓ Boots directly from BIOS (real hardware or emulator)
✓ Clears screen to black
✓ Prints "pxOS v1> " welcome message
✓ Interactive shell:
  - Echoes characters as you type
  - Handles Enter key (newline + new prompt)
  - Infinite loop (no exit - power off to quit)

WHAT MAKES IT UNIQUE
================================================================================
• Primitive-based build - Uses WRITE/DEFINE commands, not traditional assembly
• Educational - Every byte is traceable and documented
• Minimal tooling - Just Python 3, no assembler needed
• Transparent - No "magic" transformations
• Extensible - Clear path to add features

QUICK START
================================================================================
1. Boot in emulator:
   ./tests/boot_qemu.sh

2. Build from source:
   python3 build_pxos.py

3. Create bootable USB (CAUTION!):
   sudo dd if=pxos.bin of=/dev/sdX bs=512 count=1
